**AEO** stands for **Answer Engine Optimization**. Where SEO targets ranking in keyword-based search results, AEO targets being chosen as the *source of truth* in AI-generated answers.

The Dualmark framework is one implementation of an AEO strategy: serve human-friendly HTML to humans, and clean machine-friendly markdown to bots — using HTTP content negotiation, the same URL works for both.
